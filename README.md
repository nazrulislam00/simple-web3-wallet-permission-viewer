# Simple Web3 Wallet Permission Viewer

This is a beginner-friendly Web3 project.

This project helps you:
- Connect a crypto wallet
- Learn how wallet permissions work
- Display connection status on screen
- Understand wallet access flow

This project uses:
- HTML
- JavaScript
- MetaMask API

## Features
- One-click wallet connect
- Show wallet connection status
- Simple permission logic
- Clean and easy code

## Requirements
- MetaMask installed
- Internet connection
- Basic JavaScript knowledge

## How to Run
1. Download this repository
2. Open `index.html` in browser
3. Click "Connect Wallet"
4. See connection status update

## File Structure
- index.html -> Main webpage
- app.js -> App logic
- connect.js -> Wallet connect
- status.js -> Permission status
- ui.js -> Screen updates
- validate.js -> Checks
- logger.js -> Logs
- config.js -> App settings
- format.js -> Text formatting
- utils.js -> Helpers

## Learning Goal
This project helps beginners learn:
- How websites request wallet access
- How permissions work in Web3
- How wallet security works
